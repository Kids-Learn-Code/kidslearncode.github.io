Python Syntax

  1. Comments: # This is a comment
  2. Variables: x = 10
  3. Data Types: int, float, str, list, tuple, dict, set, bool
  4. Strings: text = "Hello, World!"
  5. String Concatenation: "Hello" + " " + "World"
  6. Formatted Strings (f-strings): f"Hello {name}"
  7. Arithmetic Operators: +, -, *, /, %, **, //
  8. Comparison Operators: ==, !=, >, <, >=, <=
  9. Logical Operators: and, or, not
  10. If Statements: if x > 5: print("x is greater than 5")
  11. Else Statements: else: print("x is not greater than 5")
  12. Elif Statements: elif x == 5: print("x is 5")
  13. For Loops: for i in range(5): print(i)
  14. While Loops: while x < 5: x += 1
  15. Functions: def my_function(): print("Hello!")
  16. Function Arguments: def greet(name): print(f"Hello, {name}")
  17. Return Statement: def add(a, b): return a + b
  18. Lambda Functions: square = lambda x: x**2
  19. Lists: my_list = [1, 2, 3]
  20. List Comprehensions: squares = [x**2 for x in range(10)]
  21. Tuples: my_tuple = (1, 2, 3)
  22. Dictionaries: my_dict = {"name": "Alice", "age": 25}
  23. Sets: my_set = {1, 2, 3}
  24. Try-Except: try: x = 1 / 0 except ZeroDivisionError: print("Error")
  25. Classes: class Person: def __init__(self, name): self.name = name
  26. Inheritance: class Student(Person): pass
  27. Import Statements: import math
  28. Modules: import my_module
  29. Reading Files: with open('file.txt') as f: content = f.read()
  30. Writing Files: with open('file.txt', 'w') as f: f.write("Hello")
  31. List Methods: my_list.append(4), my_list.pop()
  32. Dictionary Methods: my_dict.keys(), my_dict.values()
  33. Random Module: import random; random.choice([1, 2, 3])
  34. Date and Time: from datetime import datetime; now = datetime.now()
  35. Regular Expressions: import re; re.search('pattern', 'text')
  36. Map Function: list(map(lambda x: x * 2, [1, 2, 3]))
  37. Filter Function: list(filter(lambda x: x > 1, [1, 2, 3]))
  38. Zip Function: list(zip([1, 2, 3], ['a', 'b', 'c']))
